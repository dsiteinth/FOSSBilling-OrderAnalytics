# Changelog

All notable changes to the FOSSBilling Order Analytics module will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-10

### Added
- Initial release of the Order Analytics module.
- Real-time Date Range selector with convenient presets (Today, Last 7 Days, Last Month, etc.).
- Real-time Product Category filtering to drill down into specific product segments.
- Toggle to easily exclude or include "Pending Setup" orders.
- Comprehensive Summary cards displaying Total Orders, Active Orders, Total Revenue, Refunds, and Average Order Value (AOV).
- Percentage growth indicators comparing current performance against the previous period.
- Interactive Revenue Overview line chart for side-by-side period comparison.
- Product Categories Distribution and Top Payment Methods Doughnut charts.
- Detailed Leaderboards for "Top Selling Products" and "Top Spenders/Clients".
- Recent Orders quick-view table.
- Sleek and modern UI built on Tabler, matching the FOSSBilling admin aesthetic.
